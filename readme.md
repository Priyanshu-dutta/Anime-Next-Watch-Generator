🎌 Anime Next Watch Generator
📌 Overview

Anime Next Watch Generator is a web application designed for anime lovers who struggle to decide what to watch next. The app fetches trending and top-rated anime and allows users to explore, filter, and discover new shows effortlessly.

🎯 Purpose

With thousands of anime available, choosing the next one can be overwhelming. This project simplifies decision-making by:

Showing trending anime
Allowing genre-based filtering
Providing quick access to anime details and synopsis
🌐 API Used

This project uses the Jikan API, an unofficial REST API for MyAnimeList.

🔗 API Documentation: https://docs.api.jikan.moe

✨ Features
🔥 Trending Anime List
Displays currently trending/popular anime
🎭 Genre-Based Filtering
Users can filter anime by genres (Action, Romance, Comedy, etc.)
⭐ Top Rated Section
Browse highly rated anime easily
🔍 Search Functionality (Planned)
Search anime by name
📖 Expandable Synopsis Card
Click to view more details about each anime
📊 Sorting Options (Planned)
Sort by rating, popularity, or release date
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript, react
API Integration: Jikan API
Version Control: Git & GitHub
